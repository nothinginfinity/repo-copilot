Setup

Live in 15 minutes

1

Create your Notion database

Duplicate the included database template into your Notion workspace. Add your GitHub repo URLs to the GitHub URL field.

2

Connect your GitHub

Fork the repo-copilot template. Add your Notion token as a GitHub secret. Edit two lines in the workflow file with your username and DB ID.

NOTION_DB_ID: abc123...
GH_ORG: yourusername
3

Run the sync

Trigger the workflow from GitHub Actions. Watch live URLs populate in Notion. Enable GitHub Pages on each repo and your apps go live.
